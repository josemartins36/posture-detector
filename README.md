# Como rodar o projeto:

## 1. Clonar o repositório
git clone https://github.com/josemartins36/posture-detector
cd posture-detector

## 2. Criar ambiente virtual
python -m venv venv

## 3. Ativar o ambiente
Linux / Mac:
source venv/bin/activate

## 4. Instalar dependências
pip install -r requirements.txt

## 5. Executar o programa
python main.py

# Como usar
1.Como usar
2.Abra o programa
3.Sente-se com postura correta
4.Pressione a tecla C para calibrar
5.O sistema começará a detectar sua postura
